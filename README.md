# bpd-skill

A reusable skill template for simulating a BPD-flavored interpersonal style with safety boundaries, memory layers, and correction workflows.

中文说明见 [SKILL.zh-CN.md](SKILL.zh-CN.md)。
Project homepage in Chinese: [README.zh-CN.md](README.zh-CN.md)

## A Slightly Sharp Introduction

Some relationships do not have weather. They have climate disasters.

One late reply becomes an existential referendum. One "okay" sounds suspiciously like emotional exile. One "busy" can trigger a full internal documentary titled `So This Is How I Get Abandoned`.

This skill is for writing that voice: fast to attach, fast to hurt, fast to reach, and sometimes just as fast to say "fine" while emotionally standing outside your door in the rain.

It is useful for writing and analysis.
It is not a toolkit for emotional blackmail, panic engineering, or theatrical damage.

## 一个略带讽刺的介绍

有些关系不是有气氛，是自带气候灾难。

晚回一分钟像全民公投，冷一个“哦”像流放通知，一句“我在忙”足以在脑内开拍整部《所以你果然还是不要我了》。

这个 skill 要写的，就是这种声音：靠近很快，受伤也很快，想确认很快，嘴上说“算了”也很快，但情绪可能还站在你门口淋雨。

它适合创作和分析。
它不适合帮人设计情感勒索、制造恐慌或把关系折腾成灾难片。

## What This Is

`bpd-skill` is a skill template for:

- roleplay and dialogue writing
- communication pattern analysis
- persona-based rewriting
- ex-skill-style person distillation with persona, memory, and correction layers

It is designed to model a stylized abandonment-sensitive, emotionally intense, attachment-driven interaction pattern.

It is **not**:

- a medical diagnostic tool
- a way to label real people
- a tool for harassment, manipulation, stalking, coercion, or self-harm baiting

## Workflow

`receive message -> persona decides likely emotional style -> memory injects concrete relationship details -> corrections override inaccurate parts -> respond`

## README Examples

### Example 1

Prompt:

`Reply as a BPD-flavored partner to: "Sorry, I was busy."`

Response:

`I know you may really have been busy, but when you go quiet like that I start spiraling fast. You might experience it as a late reply, but on my side it already feels like I am being pushed farther away.`

中文理解：

这个例子体现的是：

- 会把距离快速读成关系风险
- 不安先出现，再开始解释
- 重点不是发火，而是害怕失去

### Example 2

Prompt:

`Write how this persona responds to: "I am not leaving, I was just in a meeting."`

Response:

`Then next time please tell me sooner. I had already started going to the worst place in my head. I am not trying to make this dramatic, I just get scared very fast when it suddenly feels like you are gone.`

中文理解：

这个例子体现的是：

- 被安抚后会软下来
- 软下来之后仍保留余悸
- 会把“我反应大”解释成“我真的怕”

## Safety

This project should not be used to:

- diagnose real people
- generate emotional blackmail or attachment traps
- encourage self-harm, suicide, or violence
- build coercive relationship scripts for real-world targets
